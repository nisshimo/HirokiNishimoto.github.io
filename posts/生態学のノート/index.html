生態学

生態学とはどんな学問か?

生態学とは, 生物の生活の法則を, その環境との関係で解き明かす科学である.

  2. 自己複製性
  3. 自己維持性

  • 生態系多様性
  • 種多様性

進化から見た生態

進化とは何か?

最もミクロな時間スケールでの「生物の進化」の定義は

集団内での遺伝子頻度の系図の変化

これはすごく地味な変化だが、この地味なが時々刻々積み重なることで、やがて栄子盛衰の大きな変遷をもたらす.

突然変異と修復機構

DNAの突然変異の要因
  • 内的要因: 複製エラー, 酸素ラジカル
  • 外的要因: 放射線, 化学物質など/

修復機構がはたらけばDNAは正常に戻るが、そうでないと突然変異となり、生物の進化、個体の分化につながる.

生態学の2つの問い

  1. How 至近要因: 遺伝的基盤、生理
  2. Why 究極要因: 自然選択(適応度)

多くの進化生態学の研究ではこの2つの問いで研究していくことが重要. Whyに答えるのが自然選択(適応度)

自然選択

自然選択の基本原理

以下の３つの条件が揃えば、左様は自律的に進む

  • 個体間に変異あり
  • その変異は遺伝する
  • 変異に応じて繁殖・生存に有利不利

自然選択の3つのかかり方

  • 安定的選択
  • 方向性選択
  • 分断化選択、多様化選択

自然選択の事例

  • ダーウィンフィンチの嘴の多様性
  • サンザシミバエの系統分化
  • オオシモフリエダシャクの工業暗化
  • 殺虫剤抵抗性の増進
  • タンガニーカ湖のカワスズメ科の鱗食い魚
  • マイマイの巻き型
  • ヒマラヤを超えて渡るインドガン
  • キリンの首の進化

現代進化生物学の2つの理論的支柱

進化の主な要因は何か？

総合説(自然選択説)

突然変異 ＋ 方向性選択(適用度の高い変異体を増やす作用) => 適応進化

量的形質の進化

総合説では、選択差が自然選択の強さの尺度になる.

選択差 = 選択前/後の集団の形質平均値の差

選択反応 = 1世代を経た集団同士の形質平均値の差

遺伝率 $h^2 = \frac{R}{S}$

中立説

突然変異 + 遺伝的浮動 + 純化淘汰(適応度の低い変異体を足切りする作用) => 分子進化

突然変異には、自然選択上中立な変異が多いはず。

誘致な突然変異はごく小数. 大部分は有害か中立となる.有害変異は早々に消滅. 残る大部分は中立変異.

突然変異による変異供給と遺伝的浮動による固定-> 置換速度

分子に起こる置換率（進化速度）は, 中立な突然変異の発生率に等しい.

完全中立だったら, $2N * uf * \frac{1}{2N} = uf$

但し, $N$ : 集団の個体数 $u$ : 突然変異発生率 $f$ : そのうち中立である割合

一定速度で分子情報は置き換わる.

遺伝的浮動
モラン過程

新しい対立遺伝子は, 有限時間内に消滅or固定

完全中立の場合は, 出現から平均4N世代で固定する

中立説を支持する例
  1. 一定速度で分子情報が置換
  2. タンパク質をコードする遺伝子では, 同義置換は非同義置換よりも圧倒的に早い
  3. 偽遺伝子, イントロンなど, 機能を持たないDNA領域は高速で置換

種分化

種分化とは?

今まで同種だった集団感に生殖隔離が生じて互いに交配しなくなり、新しい種が形成される過程

生殖隔離の２つの過程

  1. 交配前隔離

2つの集団間で活動場所や住み場所に微妙に違いがあるため交配に至らない場合.

  1. 交配後隔離

交配しても稔性のある子が生まれない.

種分化の2つのプロセス

  • 異所的種分化
それまで同種だった集団が、移住や地形の変化などにより地理的に別々の場所に離され(地理的隔離),そこで独自の遺伝的分化が進むうちに生殖的隔離が達成される過程.

少数者の移住 もとの集団の遺伝的構成の一部だけが新たな生息地に移住した集団の一部に引き継がれるため、移住が達成された時点ですでに遺伝的構成に差異が生じている.(創始者効果)

  • 同所的種分化
同じ地域に住んでいながら(地理的隔離なし), 活動時間や住み場所の微妙な違いによって集団が徐々に分化して交配前隔離が進み, 生殖隔離が達成される過程.

分断化選択が生じることが必要

交雑個体, 中間型が不利になり, 種分化が進む.

生態的同位種 … 自然選択を裏付け

生活史の適応進化

生活史の適応とは

生物のもつ携帯や行動、生理的、生態的形質が、その環境下で生活するのにうまく理にかなっていること. 繁殖や生存スケジュールもその対象.

適応は進化現象の1つ.自然選択がこれをもたらす.

r/K選択

繁殖のコスト

繁殖に多くのエネルギーを割けば, 自らの生存には少量のエネルギーしか回せなくなる.

r選択
  • 不安定に多きく変動する過酷な環境 => 個体密度小, 増殖力が有利
  • 小卵多産への進化

  • 産み放し
  • 繁殖投資率が大

K選択
  • 安定または周期的に変化する温暖な環境 => 個体密度大, 競争力が有利(環境収容力には限界)
  • 大卵少産への進化
  • 遅い成長, 遅い繁殖
  • 親による子の保護
  • 繁殖投資率が小
  • 多回繁殖

多種共存を説明する2つの理論

平衡理論としての「群衆理論」

仮定

  • 種間競争を重視
  • 群衆は平衡状態
  • 資源利用パターンの重複度 = 種間競争の強さ

=> 共存にはある程度以上の資源利用パターンの分割が必要

=> 棲み分け, 食い分け

種間競争の厳しい状況では共存可能なニッチ重複度が小さくなる この説に立脚した生物群集の捉え方を特に群衆理論と呼ぶ.

非平衡理論としての「非平衡共存説」

自然界では本当に種間競争が卓越?

  • ランダム帰無モデル

非平衡である状況のほうが一般的?

自然界は群衆理論が想定したような平衡状態にはなく, むしろ非平衡状態において群集構成種は共存していると主張

現在の群集の多様性の捉え方

=> 多元的な解釈

  • 植物性昆虫では非平衡共存が支持される例が多い

  • 栄養段階の上位の種では種間競争が強く作用する.
